# Air-Bnb
Title: Airbnb Data Analysis & Price Trend Prediction 🏠📈

Description:
A comprehensive exploratory data analysis (EDA) project focused on uncovering trends in Airbnb accommodations, pricing, and guest reviews.

🛠️ Tools & Libraries Used:

1. Python: The backbone of the analysis.

2. Pandas: Used for robust data manipulation, handling missing records, and complex data cleaning.

3. Matplotlib & Seaborn: Used to generate insightful, high-quality data visualizations.

🔍 Key Analyses Performed:

1. Price Analysis: Evaluated pricing trends across various room types to establish baselines for price prediction.

2. Accommodation Popularity: Mapped out guest preferences based on location, room type, and overall popularity.

3. Availability Tracking: Visualized the availability of different room types throughout the dataset.

4. Review Timelines: Created time-series charts to analyze how review volumes change over time.
